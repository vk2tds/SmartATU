# SmartATU
A smart automatic HF antenna tuner, including a digital radio interface so that the tuner will tune the antenna without needing to actually transmit.

There is a major hole in the market for antenna tuners. Devices like the ATU-100 exist, with amazing capabilities. However, these units are essentially standalone, with the possible ability to trigger a tuning cycle from the radio and have the tuner indicate that tuning has been complete. Units like the LDG AT-1000ProII do come with an optional USB interface. The capabilities are quite limited.

# Requirements

* Relay switched capacitor and inductor antenna tuner with SWR monitoring
* Input/Output for ICOM external tuner connector
* Digital interface to the radio using the CAT interface
* Ability to control and monitor via USB/Ethernet
* Optional RF measurement chip
